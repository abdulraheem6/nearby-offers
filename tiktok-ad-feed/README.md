# TikTok-Style Ad Feed

A mobile-first, TikTok-style vertical feed for ads, built with React and Vite. Fully deployable on GitHub Pages.

## Features

✅ Vertical swipe feed with snap scrolling  
✅ Full-screen cards  
✅ Auto-play videos when visible (IntersectionObserver)  
✅ Images and videos mixed in same feed  
✅ YouTube support  
✅ `ads.json` driven content  
✅ Featured ads with badge  
✅ Category filters (All, Electronics, Food, Home, Fashion, Beauty, Auto)  
✅ WhatsApp CTA button on each ad  
✅ Infinite scroll simulation  
✅ GitHub Pages deployment ready  

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
npm run dev
